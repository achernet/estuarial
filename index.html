Estuarial¶

Estuarial is an open-source API for Thomson Reuters’ QA Direct. The aim of Estuarial is help quants, financial researchers, and other domain experts fetch data from the various DBs within QAD in a complete, expressible, and efficient manner through Python.

To get started quickly, follow the Quickstart.

Visit the source repository: https://github.com/ContinuumIO/estuarial
